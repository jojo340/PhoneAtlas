# Phone Atlas Support

Thank you for using Phone Atlas.

If you run into an issue, find incorrect phone specifications, or want to suggest new models for the catalog, contact the developer:

- Email: yangnote@qq.com
- GitHub Issues: https://github.com/jojo340/PhoneAtlas/issues

## FAQ

### What should I do if catalog update fails?

Please make sure your network can access GitHub or the configured remote `phones.json` source. If the issue continues, try again later.

### Are all specifications guaranteed to be accurate?

Phone specs are compiled from public information and maintained through the remote catalog. Specifications may vary by country, carrier, storage configuration, and sales version. Please confirm important details with the official product page or actual retail version.

### How can I delete local data?

Deleting the app removes local favorites, comparison list, language settings, and cached catalog data.
